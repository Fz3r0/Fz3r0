### Cambiar IP a interfaces (DHCP/Est치tica)

- Es importgante cambiar las `x.x.x.x` por la IP fija que queremos.

- El gateway se debe usar el real del router del laboratiorio, ej. `192.168.1.254` "Telmex"

